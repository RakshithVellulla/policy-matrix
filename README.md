# Policy Matrix

Policy Matrix is an AI-powered healthcare policy comparison platform.

## Features

- Upload two healthcare policy PDFs
- Extract text from PDF documents
- Compare policy differences
- Generate AI-powered summaries using Google Gemini
- Download analysis reports as PDF

## Technologies

Frontend
- React
- TypeScript
- Tailwind CSS

Backend
- FastAPI
- Python

AI
- Google Gemini API

PDF
- PyMuPDF
- jsPDF

## Installation

Frontend

npm install
npm run dev

Backend

pip install -r requirements.txt
uvicorn main:app --reload
